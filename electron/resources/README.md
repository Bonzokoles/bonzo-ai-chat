# Ikona info
# Potrzebujesz ikony icon.ico w tym folderze

Do wygenerowania ikony możesz użyć:
- https://icoconvert.com/ (PNG → ICO)
- https://www.favicon-generator.org/ (multiple sizes)

Wymagane rozmiary:
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

Skopiuj plik icon.ico tutaj: electron/resources/icon.ico
